# Misskey Hub
Misskey Hub is an Official Site of Misskey.

Built with [VuePress](https://github.com/vuepress/vuepress-next).

## Develop
Node.js and npm are required.

### Launch dev server
```
npm run dev
```

### Build as static site
```
npm run build
```
